# Zeda.io Clone - AI-Powered Development Platform

A pixel-perfect, fully functional clone of [zeda.io](https://zeda.io/) built with Next.js, Tailwind CSS, and TypeScript.

## 🚀 Features

- **Pixel-Perfect Design**: Exact replication of zeda.io's UI/UX design
- **Responsive Layout**: Flawless experience across mobile, tablet, and desktop
- **Interactive Elements**: Smooth animations, hover effects, and scroll interactions
- **Modern Tech Stack**: Built with Next.js 14, Tailwind CSS, and TypeScript
- **Component-Based Architecture**: Clean, reusable React components
- **Performance Optimized**: Fast loading and smooth animations
- **Production Ready**: Ready for deployment on Vercel

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Fonts**: Inter & JetBrains Mono (Google Fonts)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Hackwave-webApp
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
npm start
```

## 🎨 Customization

### Colors
The project uses a custom color palette defined in `tailwind.config.js`:
- **Primary**: Blue tones for main elements
- **Secondary**: Gray tones for text and backgrounds
- **Accent**: Purple tones for highlights

### Typography
- **Inter**: Main font for body text
- **JetBrains Mono**: Monospace font for code elements

### Animations
Custom animations defined in Tailwind config:
- `fade-in`: Smooth opacity transitions
- `slide-up`: Upward slide animations
- `slide-down`: Downward slide animations
- `scale-in`: Scale and fade effects
